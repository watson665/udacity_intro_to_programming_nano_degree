What do the sub and sup elements do?
They move the text vertically, as subscript or superscript. Originally, HTML started out as a tool for scientific and academic work; and these are really useful for science and math, like writing x2+3x+4 = 1 or chemical formulas like H2O.

2. When you leave off a closing tag, how does the browser know when to close that element?
When you leave off an optional closing tag, it is still implied—that is, the browser will figure out where the element should be closed, even though you didn't explicitly tell it. The browser knows there has to be a closing tag when it gets to the start tag of a new element that can't be inside of the last one. For example, when it gets to the start of a new p element, it knows the previous p element must be closed.
